## Steps in git

1. Clone Project
```bash
$ git clone https://github.com/IzzleNizzle/testingGit.git
```
2. Create your own branch
``` git checkout -b branch-name```
3. Make changes
4. Push changes to your branch
```git add .```
```git commit -m 'Some message"```
```git push```
```git push --set-upstream origin branch-name```
